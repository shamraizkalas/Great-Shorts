## 📥 انسٹالیشن کی ہدایات

### مرحلہ 1: APK ڈاؤن لوڈ کریں
- GitHub Releases سے APK فائل ڈاؤن لوڈ کریں

### مرحلہ 2: نامعلوم ذرائع کی اجازت دیں
1. اپنے فون میں **Settings** کھولیں
2. **Security** یا **Privacy** میں جائیں
3. **Unknown Sources** یا **Install unknown apps** کو **ON** کریں

### مرحلہ 3: APK انسٹال کریں
1. ڈاؤن لوڈ شدہ APK فائل پر کلک کریں
2. **Install** دبائیں
3. انسٹالیشن مکمل ہونے کا انتظار کریں

### مرحلہ 4: ایپلیکیشن شروع کریں
- ایپلیکیشن کو کھولیں اور لطف اندوز ہوں!
